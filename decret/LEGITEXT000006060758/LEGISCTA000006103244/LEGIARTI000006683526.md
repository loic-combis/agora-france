### Article 64

[Voir sur Légifrance](https://www.legifrance.gouv.fr/loda/article_lc/LEGIARTI000006683526/)

Les participations incombant à l'Etat, d'une part, aux collectivités locales, d'autre part, dans les dépenses d'aide sociale sont fixées comme suit : DEPARTEMENT : Guadeloupe GROUPE I : Etat : 92 Département : 8. GROUPE II : Etat : 84 Collectivités locales : 16. GROUPE III : Etat : 68 Collectivités locales : 32. DEPARTEMENT : Guyane et Inini GROUPE I : Etat : 96 Département : 4. GROUPE II : Etat : 92 Collectivités locales : 8. GROUPE III : Etat : 84 Collectivités locales : 16. DEPARTEMENT : Martinique GROUPE I : Etat : 92 Département : 8. GROUPE II : Etat : 84 Collectivités locales : 16. GROUPE III : Etat : 68 Collectivités locales : 32. DEPARTEMENT : Réunion GROUPE I : Etat : 92 Département : 8. GROUPE II : Etat : 84 Collectivités locales : 16. GROUPE III : Etat : 68 Collectivités locales : 32.
