### Article 1

<!-- CID: LEGIARTI000006728431 — version: LEGIARTI000006728431 — état: VIGUEUR — en vigueur: 1967-12-09 → 2999-01-01 -->
[Voir sur Légifrance](https://www.legifrance.gouv.fr/loda/article_lc/LEGIARTI000006728431/)

Les billets de la Banque de France de :

500 anciens francs, type 1953 ;

1.000 anciens francs, type 1945 ;

1.000 anciens francs, type 1953 ;

5.000 anciens francs, type 1949 ;

5.000 anciens francs, type 1957 ;

10.000 anciens francs, type 1945 ;

5 nouveaux francs, type 1959 ;

10 nouveaux francs, type 1959 ;

50 nouveaux francs, type 1959, cesseront d'avoir cours légal et pouvoir libératoire à dater du 1er avril 1968.
