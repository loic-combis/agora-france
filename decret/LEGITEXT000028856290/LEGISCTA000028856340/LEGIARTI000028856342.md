### Article 29

<!-- CID: LEGIARTI000028856342 — version: LEGIARTI000028856342 — état: VIGUEUR — en vigueur: 1956-01-01 → 2999-01-01 -->
[Voir sur Légifrance](https://www.legifrance.gouv.fr/loda/article_lc/LEGIARTI000028856342/)

Le personnel d'exécution et de service est réparti d'après leurs fonctions dans les groupes suivants : 1er groupe : commis, agent responsable de la surveillance ; 2e groupe : sténodactylographe, préposé téléphoniste, chef magasinier ; 3e groupe : surveillant, magasinier, agent de bureau ; 4e groupe : gardien, agent de service de 1re catégorie ; 5e groupe : manutentionnaire, agent de service de 2e catégorie.
