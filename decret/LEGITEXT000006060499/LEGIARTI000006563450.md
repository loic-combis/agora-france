### Article 2

<!-- CID: LEGIARTI000006563450 — version: LEGIARTI000006563450 — état: MODIFIE — en vigueur: 1955-01-01 → 2012-01-01 -->
[Voir sur Légifrance](https://www.legifrance.gouv.fr/loda/article_lc/LEGIARTI000006563450/)

Les indemnités représentatives de frais se divisent en : Indemnité pour charges aéronautiques ; Indemnité pour frais de représentation ; Indemnité spéciale aux territoires du Sud ; Indemnité forfaitaire ; Indemnité de première mise d'équipement ; Indemnité de première mise de harnachement ; Indemnité pour changement d'uniforme ; Indemnité pour perte d'effets ; Indemnité de départ ; Indemnité spéciale d'alimentation ; Indemnité d'habillement.
