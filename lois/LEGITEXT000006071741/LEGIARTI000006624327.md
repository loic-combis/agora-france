### Article 8

<!-- CID: LEGIARTI000006624327 — version: LEGIARTI000006624327 — état: ABROGE — en vigueur: 1888-03-02 → 1985-05-24 -->
[Voir sur Légifrance](https://www.legifrance.gouv.fr/loda/article_lc/LEGIARTI000006624327/)

Les poursuites sont portées devant le tribunal de police correctionnelle dans le ressort duquel est situé le port où les contrevenants ont été conduits. Le tribunal statue dans le plus bref délai possible.
