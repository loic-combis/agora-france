Programme de l'examen professionnel pour l'accès aux emplois de chauffeur de chaudières à haute pression dans les établissements d'hospitalisation, de soins ou de cure publics. A. - Epreuves théoriques. La chaleur : - Notions de quantité de chaleur ; - Pouvoir calorifique ; - Chaleur latente ; - Chaleur sensible ; - Température ; La combustion : - Le combustible : éléments combustibles, carbone et hydrogène. Le comburant : - Composition de l'air atmosphérique, oxygène, azote. Les produits de combustion : - Solides : cendres, mâchefers, suie ; - Gazeux : CO2, oxygène, azote, vapeur d'eau. - Combustion complète. - Combustion incomplète. - Chaleur disponible : Pouvoir calorifique. - Chaleur utilisée : rendement. - Chaleur perdue : dans les résidus solides, chaleur latente dans les fumées, chaleur sensible. - Chaleur perdue : par radiation. - L'excès d'air : chaleur perdue par excès d'air. Contrôle de l'excès d'air. Estimation des pertes par résidus solides et gazeux. Estimation du rendement. Les combustibles (caractéristiques d'utilisation) : Combustibles solides : - la granulométrie, - l'eau accidentelle, - les matières volatiles, - la cokéfaction et agglutination, les cendres (pourcentage et fusibilité), - le soufre nuisible, - le pouvoir calorifique ; Combustibles liquides : - la viscosité de pompage, - la viscosité de pulvérisation, - la densité, - le soufre nuisible, - le pouvoir calorifique ; Combustibles gazeux ; - pouvoir calorifique ; - composition des gaz industriels. Réglage de la combustion : - Débit de combustible ; - Débit d'air de combustion : air neutre et excès d'air ; - Evacuation des fumées : fumées neutres et excès d'air ; tirage ; - Contrôle de la fumivorité : pollution atmosphérique. La vapeur d'eau : - Pression ; - Température d'ébullition de l'eau ; - Vapeur saturée ; - Surchauffage. L'eau d'alimentation : - Epuration ; - Contrôle. L'eau de chaudière : - Contrôle ; - Extraction La sécurité : - Manque d'eau ; - Indicateur de niveau d'eau ; - Appareils d'alarme ; - Clapet de retenue d'alimentation ; - Pression : manomètre, soupapes de sécurité ; - Allumage d'une unité : mise en pression (précautions à prendre) ; - Mise ne veilleuse ; - Extinction. B. - Epreuves pratiques. I. - Observations des règles de sécurité. II. - Conduite des feux : - Maintien de l'allure ; - Contrôle de l'excès d'air ; - Contrôle de la fumivorité. III. - Travaux d'entretien : - Ramonage ; - Niveau d'eau ; - Contrôle de l'eau d'alimentation de l'eau des chaudières ; - Extraction ; - Réfection d'un joint.
